# SecCloudPath website

Minimal, modern business website scaffold for **SecCloudPath**.

## Tech

- Next.js (App Router) + React + TypeScript
- Tailwind CSS

## Getting started

Install Node.js (LTS), then:

```bash
npm install
npm run dev
```

## Routes

- `/` Home
- `/services`
- `/about`
- `/contact`

Locale prefixes: `/no` (default) and `/en` (see `middleware.ts`).

## Deploy (go online)

See **[DEPLOY.md](./DEPLOY.md)** — GitHub + Vercel step-by-step.

Quick checklist: install **Git** → create **GitHub** repo → `git push` → **Import** repo on [vercel.com/new](https://vercel.com/new).
